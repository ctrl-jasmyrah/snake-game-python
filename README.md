# Snake Game 🐍

A recreation of the classic Snake game using Python.

## Features
- Snake movement
- Food generation
- Score tracking
- Collision detection

## Concepts Used
- Python
- OOP
- Turtle Graphics

## Controls
Arrow Keys → Move Snake
